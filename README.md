# junit4-junit5

Welcome to junit4-junit5! This is a sample project for using JUnit 4 alongside JUnit 5. Read the accompanying blog post at https://blog.codecentric.de/2018/09/alt-trifft-neu-junit-4-5-in-einem-projekt/.
